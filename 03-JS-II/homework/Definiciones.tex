 && (AND): es un operador LOGICO que evalua ambas expresiones y devuelve el valor TRUE solo si ambas variables son TRUE.
 Si una de las variables es false y true este devoolvera FALSE.

 Demostracion mediante uan tabla guia:
 VARIABALE1          VARIABALE2        VARIABLE1 && VARIABLE2
   TRUE                TRUE                     TRUE
   TRUE                FALSE                    FALSE
   FALSE               TRUE                     FALSE
   FALSE               FALSE                    FALSE

 OPERADOR || (OR): A diferencia de && este operador devuelve TRUE si alguno de los operadores es TRUE. 

 VARIABALE1          VARIABALE2        VARIABLE1 && VARIABLE2
   TRUE                TRUE                     TRUE
   TRUE                FALSE                    TRUE
   FALSE               TRUE                     TRUE
   FALSE               FALSE                    FALSE 

   OPERADOR ! (NOT) Es un operador logico mas usado en la negacion, se usa para devolver el valor contrario de la variable.}
